We need a first-time installer/setup wizard for lxp-llm-gateway.

Goal:
Make the project easy to install and configure like a serious self-hosted product.

Target flow:
1. docker compose up
2. open admin UI
3. first-time setup wizard appears
4. create super admin
5. create first tenant
6. configure providers
7. test provider credentials
8. optionally create OpenWebUI integration client/API key
9. finish setup and lock setup mode

Important security rules:
- Setup wizard is available only when no super admin exists and installation_state is not completed.
- Add a setup token mechanism to prevent public unauthenticated setup takeover.
- Setup must be disabled permanently after completion.
- Technical secrets such as encryption key/JWT secrets should come from env or generated by install script, not stored insecurely by the wizard.
- Never log provider API keys or generated secrets after display.

Backend tasks:
- Add InstallationState entity/table.
- Add setup status endpoint.
- Add setup bootstrap endpoint guarded by setup token.
- Create first super admin.
- Create first tenant.
- Create initial tenant membership.
- Configure tenant provider settings.
- Store provider credentials using existing encrypted credential storage.
- Create optional integration client/API key.
- Mark installation completed.

Frontend tasks:
- Add /setup route.
- Add multi-step wizard:
  - System Check
  - Super Admin
  - First Tenant
  - Providers
  - Policies
  - Integration Clients
  - Finish
- Add clear explanations for each step.
- Add provider credential test buttons.
- Add mobile-friendly layout.

CLI/script tasks:
- Add install/bootstrap script to generate .env values.
- Validate required env vars.
- Start docker compose.
- Print setup URL and setup token.

Acceptance criteria:
- Fresh install guides the user from empty system to usable admin dashboard.
- Existing installations never show setup wizard.
- Setup cannot be re-run after completion unless explicitly reset by CLI/admin maintenance command.
- OpenWebUI can be configured from the wizard without manual tenant/API key setup.
